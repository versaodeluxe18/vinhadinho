> **[SOURCE-AVAILABLE SERVERLESS GAME PLATFORM]** Mais do que um jogo, este projeto foi concebido como um ecossistema colaborativo e extensível. Desenvolvedores são convidados a criar novos modos de jogo, implementar mecânicas próprias e explorar estratégias independentes de monetização associadas ao jogo, sob os termos da Versão Deluxe. Aqui, a criatividade é tratada como ativo estratégico para expansão contínua da plataforma, mantendo a propriedade intelectual e comercial restrita ao ambiente de produção da empresa.  
>  
> Este ecossistema foi arquitetado para operar com **custo zero de produção e manutenção**, eliminando dependência de servidores dedicados e reduzindo complexidade operacional. A autenticação é gerenciada via Firebase, enquanto o backend é implementado em modelo serverless utilizando Google Apps Script, com persistência e administração de dados através de Planilhas Google (TSV).  
>  
> A distribuição ocorre de forma totalmente estática, permitindo publicação em plataformas como **GitHub Pages, Google Sites, Blogger ou WordPress**, garantindo portabilidade, replicabilidade e sustentabilidade econômica da infraestrutura.

## MODO DE JOGO 

* **COLETA:** O Modo Coletar é uma experiência rápida e dinâmica, ideal para quem gosta de ação constante. Nesse modo, o jogador entra em um cenário onde o objetivo principal é coletar o máximo de itens possível, evitando obstáculos e administrando o tempo com precisão.<br><br>
* **ACAMPAMENTO:** O Modo Acampamento oferece uma jogabilidade mais estratégica e relaxante, simulando a rotina de montar e administrar um pequeno acampamento. Aqui, o foco não é velocidade, mas sim organização, cuidado e evolução do ambiente.<br><br>
* **CABANA:** O Modo Cabana é uma experiência focada na personalização e aconchego. Seu objetivo é construir, decorar e melhorar a sua cabana, deixando-a cada vez mais completa, bonita e funcional.<br><br>

## ESTRUTURA DE DIRETÓRIOS

O projeto foi organizado para ser (MPA)-Aplicação de Múltiplas Páginas onde cada arquivo da pasta game é uma pagina.

```text
📦 vinhadinho
 ┣ 📜 index.html------------------# Inicio.
 ┣ 📂 game/-----------------------# Códigos do jogo
 ┣ 📂 doc/------------------------# Documentação
 ┗ 📂 arquivos/-------------------# Arquivos e base64
```

## SISTEMA DE APOIO E DOAÇÕES

Todas as formas de monetização do VINHADINHO seguem o mesmo princípio:

O jogador realiza uma doação voluntária,
recebe uma contrapartida simbólica ou funcional dentro do jogo,
e contribui diretamente para a evolução estrutural do projeto. Descubra mais na documentação. <br><br>
